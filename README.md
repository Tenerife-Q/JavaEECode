# JavaEECode

面向 Java 基础与进阶的示例集合，涵盖面向对象、包管理、多态、优先级、进阶语法与练习项目等。

## 目录概览
- `JavaAdvanced/`：Java 进阶示例（集合、IO、线程、异常等，可补充具体内容）。
- `JavaAdvancedAgain/`：进一步的高级示例与练习。
- `JavaLearning/`：基础语法与入门示例。
- `OOP/` 与 `OOP2/`：面向对象核心示例（类、继承、封装、多态）。
- `TestHeritage/`：继承与组合相关练习。
- `linling/`：临时/零散练习代码。
- `package/`：包与可见性示例。
- `poly/`：多态相关示例。
- `priority/`：优先级或调度相关示例。
- `README.md`：仓库说明（当前文件）。

## 快速开始
1) 克隆仓库：
   ```bash
   git clone https://github.com/Tenerife-Q/JavaEECode.git
   cd JavaEECode
   ```
2) 使用 IDE（IntelliJ IDEA / Eclipse）或命令行编译运行：
   ```bash
   javac path/to/File.java
   java com.example.Main
   ```
   > 注意：根据具体包名与目录调整编译路径。

## 如何阅读代码
- 从 `JavaLearning/` 了解基础语法，再进入 `OOP/`、`OOP2/` 熟悉面向对象。
- 在 `poly/`、`priority/`、`package/` 中查看进阶概念的独立示例。
- `JavaAdvanced/` 与 `JavaAdvancedAgain/` 可作为集合、异常、线程、IO 等专题的练习场。
- `TestHeritage/` 关注继承层次的实现与测试。
